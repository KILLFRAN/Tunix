Tunix — десктопный VPN‑клиент с современным интерфейсом. Поддерживает протоколы VLESS (на базе xray‑core) и Hysteria/Hysteria2 (на базе sing‑box).

Основные возможности:
Импорт ключей (vless://, hysteria:// и др.) и подписок (URL, JSON, base64, текст);
Группировка серверов по подпискам, автообновление подписок;
Замер задержки (ping) разными способами, сортировка серверов;
Настройка системного прокси либо работа через локальный SOCKS5/HTTP;
Тёмная и светлая темы, поддержка русского и английского языков;
Отображение статуса подключения, аптайма, пинга, данных о ПК;
Сохранение настроек и списка серверов.

Платформы: Windows,Android
